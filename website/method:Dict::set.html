Dict::set

Return a new dict with key associated with value.

Exampleslet scores = Dict["Mario" => 10]
scores.set("Mario", 11) //-> Dict["Mario" => 11]

scores.set("Luigi", 8)
//-> Dict["Mario" => 10, "Luigi" => 8]
