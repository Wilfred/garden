# `<`, `<=`, `>` and `>=` Operators

`<` evaluates to `True` if the first integer is strictly less than the
second.

```
1 < 2 //-> True
10 < 10 //-> False
100 < 99 //-> False
```

`<=` evaluates to `True` if the first integer is less than or equal to
the second.

```
1 <= 2 //-> True
10 <= 10 //-> True
100 <= 99 //-> False
```


`>` evaluates to `True` if the first integer is strictly greater than
the second.

```
2 > 1 //-> True
10 > 10 //-> False
99 > 100 //-> False
```

`>=` evaluates to `True` if the first integer is greater than or equal
to the second.

```
2 >= 1 //-> True
10 >= 10 //-> True
99 >= 100 //-> False
```
