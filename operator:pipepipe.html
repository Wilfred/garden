|| Operator

|| performs logical OR of two Bool values. See also &&.

True || True   //-> True
True || False  //-> True
False || True  //-> True
False || False //-> False

Note that || does not currently use short-circuiting, so both sides are evaluated.

// Prints "hello"
False || fun() { 
  println("hello")
  False
}()
