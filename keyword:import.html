import keyword

You can use import to load definitions from another file.

File: greetings.gdn
public fun say_hello() {
  println("Hello!")
}
Example 1
import "./greetings.gdn" as g

g::say_hello()

You can also load all the definitions into the current file.

Example 2
import "./greetings.gdn"

say_hello()
