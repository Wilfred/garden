dbg

Write an arbitrary value to stdout, along with debugging metadata.

Returns the input, so you can use dbg in larger expressions foo(dbg(bar())).

Examples 
dbg([1, 2])
Source Code 
public fun dbg<T>(value: T): T {
  println(string_repr(value))
  value
}
