Manual

Keywords

let, fun, enum, struct, external, internal, import, if, else, while, return, test, match, break, continue, for, in, assert and as.

Built-in Types

Bool, Fun, Int, List, Namespace, NoValue, Option, Path, Result, String, Tuple and Unit.

Prelude

These functions are available in all Garden code.

dbg, error, print, println, range, set_working_directory, shell, shell_arguments, source_directory, string_repr, todo and working_directory.

Filesystem Namespace

These functions are accessible after importing __fs.gdn.

list_directory and write_file.

Example usage:

import "__fs.gdn" as fs
fs::list_directory(Path{ p: "/" })

Garden Reflection Namespace

These functions are accessible after importing __garden.gdn.

check_snippet, doc_comment_for_type, keywords, lex, namespace_functions, prelude_types and source_for_type.

Example usage:

import "__garden.gdn" as garden
garden::check_snippet("let x = 1")
