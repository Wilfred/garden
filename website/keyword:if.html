if keyword

if evaluates a conditional.

Example 1
let b = True

if b {
  print("b was true!")
} else {
  print("b was false!")
}

if is an expression in Garden, so it can be used like a ternary.

Example 2
let food = "chocolate"

let description = if food == "chocolate" {
  "tasty"
} else {
  "hmm"
}

else is optional.

Example 3
let b = True

if b {
  print("b was true!")
}
